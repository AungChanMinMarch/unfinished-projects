This project is a web application that helps a user or users to write a novel.

type - text editor (e.g. like microsoft word)
hosts on - github (most probably because it is free and easy)
database - google dev (because it is free)
production - not economic, just for fun, self use

Description
This website will be host on github and use Google Dev Firebase to save the current process.

Users can create accounts to save their works and to be able to write on multi devices. Users can create books (like google docs or facebook posts). They can share their books 
1. to let their friend or friends read a.their books b. their works(they will see everything including plot twists) or
2. to let their friend or friends comment/edit a. their novels b. their works
