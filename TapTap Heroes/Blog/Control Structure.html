Booleans & Comparisons

In Basic Concept of Python, we have seen the comparison operators (==, !=, >, <, >=, <=). They give True if the comparison statement is true. Otherwise they give False. True and False are Boolean values. There are many integer values (...,-1,0,1,2,...) But there are only two Boolean values : True and False

print(10 == 10)
print(10 == 15)
print(10 != 15)
print(10 > 15)
print(10 < 15)
print(10 >= 15)
print(10 <= 15)
